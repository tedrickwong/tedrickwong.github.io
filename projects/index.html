Reddit PC Deals Saver

A tool written in Python utilizing JSON data acquired from Reddit. This tool captures new posts on the Build a PC Sales Reddit page, analyzes the entries and separates out information (type of deal, product, price, link to page), and saves it into a database. The data is then displayed on a webpage and can be filtered by the users' preferences.
